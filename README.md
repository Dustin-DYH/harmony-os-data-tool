# 车企定制化数据分析工具

纯前端 Excel 数据分析工具，专为汽车经销商门店设计。上传通话记录 + 个微/企微客户数据，一键生成标准化效能分析报告。

## 功能特性

- **日报报表**：按门店、员工维度统计当日外呼/呼入/加微/通话时长/联系客户数等 19 项指标
- **周报报表**：当周数据汇总，维度与日报一致
- **日度趋势统计**：全量数据按日聚合，展现每日运营趋势
- **数据概览**：全局 KPI 大数卡片 + 门店运营效能总览
- **门店效能分析**：各门店客户覆盖与接通效率对比
- **员工活跃度分析**：员工多维度评分与双系统并行分析
- **时段与效率分析**：分时段/星期外呼效率，识别最佳外呼窗口
- **周度综合结论**：自动化结论与门店排名

## 数据字段

### 通话记录（必填）
- 来电时间/呼出时间
- 通话人
- 通话人所属部门
- 呼入/呼出
- 接通状态
- 手机号码
- 通话时长

### 个微客户数据（选填）
- 员工姓名、添加/上传时间、所属部门

### 企微客户数据（选填）
- 工作台姓名、添加时间、所属部门

## 核心指标

| 类别 | 指标 |
|---|---|
| 外呼情况 | 外呼量、老客跟进、新客外呼、接通量、接通率、≥30s通话、≥30s通话率、>3min深沟通 |
| 接听情况 | 呼入量、接通量、接通率 |
| 客户覆盖 | 联系客户数（呼出+呼入去重手机号） |
| 加微情况 | 个微好友、当日个微、企微好友、当日企微 |
| 沟通时长 | 平均外呼接通时长、工作手机通话总时长 |

## 技术实现

- 纯前端处理，数据不上传服务器
- 基于 SheetJS (xlsx) 读取 + ExcelJS 写入
- 字段自动识别，兼容不同表头命名
- 新客/老客精准判断（全量首次接触时间戳匹配）

## 使用方式

1. 浏览器打开 `index.html`
2. 上传通话记录 (.xlsx)
3. 可选上传个微/企微客户数据
4. 填写门店名称和日报日期
5. 点击「生成分析报告」
6. 下载 .xlsx 报告
