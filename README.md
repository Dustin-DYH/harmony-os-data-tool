# 车企定制化数据分析工具

纯前端 Excel 数据分析工具，专为汽车经销商门店设计。上传通话记录 + 个微/企微客户数据，一键生成标准化效能分析报告。

## 功能特性

- **Web 数据看板**：生成报告后自动加载 ECharts 可视化看板，支持多维度数据钻取
- **全部门店趋势概览**：所有门店外呼/接通率/加微/有效通话/平均时长趋势对比
- **分店趋势统计**：按日期+门店维度展示运营数据，含经销商筛选器
- **人员分析**：当日人员外呼、接通、加微、通话时长多维度对比
- **人员趋势分析**：按人员筛选，查看外呼量/接通率/时长/加微趋势曲线
- **多店支持**：自动识别通话人所属部门中的门店，支持经销商筛选器联动所有看板

## 数据字段

### 通话记录（必填）
- 来电时间/呼出时间
- 通话人
- 通话人所属部门
- 呼入/呼出
- 接通状态
- 手机号码
- 通话时长

### 个微客户数据（选填）
- 员工姓名、添加/上传时间、所属部门

### 企微客户数据（选填）
- 企业微信成员名、添加时间、所属部门（选填，缺失时自动通过员工姓名匹配通话记录中的门店）

## 核心指标

| 类别 | 指标 |
|---|---|
| 外呼情况 | 外呼量、老客跟进、新客外呼、接通量、接通率、≥30s通话、≥30s通话率、>3min深沟通 |
| 接听情况 | 呼入量、接通量、接通率 |
| 客户覆盖 | 联系客户数（呼出+呼入去重手机号） |
| 加微情况 | 个微好友、当日个微、企微好友、当日企微 |
| 沟通时长 | 平均外呼接通时长、工作手机通话总时长 |

## 技术实现

- 纯前端处理，数据不上传服务器
- 基于 SheetJS (xlsx) 读取 + ExcelJS 写入
- 字段自动识别，兼容不同表头命名
- 新客/老客精准判断（全量首次接触时间戳匹配）

## 使用方式

1. 浏览器打开 `index.html`
2. 上传通话记录 (.xlsx)
3. 可选上传个微/企微客户数据
4. 填写日报日期
5. 点击「生成分析报告」
6. 在 Web 看板中查看数据图表，可按经销商/日期筛选
7. 下载 .xlsx 报告
